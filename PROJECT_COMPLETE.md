## 🚀 Quick Start for New Users

### 1. Clone Repository
```bash
git clone https://github.com/dipayansardar73-decode/BHUMI_THE_FUTURE_OF_AGRITECH.git
cd BHUMI_THE_FUTURE_OF_AGRITECH
```

### 2. Setup Environment
```bash
cp .env.example .env.local
# Edit .env.local with your credentials
```

### 3. Install Dependencies
```bash
npm install
cd lambda && npm install && cd ..
cd infrastructure && npm install && cd ..
```

### 4. Deploy to AWS
```bash
cd infrastructure
export WEATHER_API_KEY=your-key-here
npx cdk bootstrap
npx cdk deploy
```

### 5. Update Environment
```bash
# Copy CDK outputs to .env.local
# Then test locally
npm run dev
```
